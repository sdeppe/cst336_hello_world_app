What is VR?

Home What is VR? What is AR? Games and Possibilities

When using Virtual Reality, the computer uses sensors and algorithms to determine the position and orientation of the users camera. The computer or device then uses this data to make it seem as the user is looking around in a simulated environment. As the user turns their head, the graphics in the program will change accordingly. Where AR tries to place virtual objects in the real world, VR instead creates a convincing and interactive world for the user.

Instead of chaning the real world to modify what the user sees in their current surroundings, VR instead attempts to create a new world and immerse the user in it. It does this by making it so the user can only see the virtual world, and by changing their view based on how their move their head.
